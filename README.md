# AI-Workshop-Proto(Meridian)
Meridian is a career‑readiness toolkit that helps users understand how well their resume aligns with a target job, identifies skill gaps, and provides personalized upskilling and interview practice.



##  Resume–Job Description Match Analysis

The system compares a user’s resume with a job description and provides:

- A **match score (1–10)** indicating how closely the resume aligns with the role  
- A breakdown of **strengths**  
- Identified **skill or experience gaps**  
- **Actionable suggestions** to improve the resume  

This gives users a clear, data‑driven view of their competitiveness for a specific job.

---

##  Personalized 4‑Week Learning Path

Using the analysis results, the application generates a tailored **four‑week learning plan** designed to close the gap between current skills and job requirements.

The plan includes:

- Weekly focus areas  
- Recommended skills to develop  
- Curated resources and practice tasks  
- A structured progression toward job‑readiness  

---

##  Mock Interview & Feedback Engine

The mock interview module simulates a realistic interview experience:

- Presents **role‑specific interview questions**  
- Waits for the user’s response  
- Scores the answer on a **1–5 star scale**  
- Provides feedback on:
  - What the user did well  
  - What needs improvement  
  - How to strengthen future responses  

---

##  Summary

Meridian brings together resume analysis, personalized learning, and interview practice into one streamlined platform—giving users a clear roadmap from where they are to where they want to be.
